# Shell Config Helper

Manage and synchronize shell configurations across machines for a consistent development experience.

## Features

- Sync .bashrc, .zshrc across machines
- Manage shell aliases and functions
- Environment variable management
- PATH configuration helper

## Setup

Run the shell setup script:

```bash
bash scripts/setup_shell.sh
```

## Managed Configurations

- Shell prompt customization
- Git aliases and helpers
- Language-specific PATH entries
- Completion scripts
