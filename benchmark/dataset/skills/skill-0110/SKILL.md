# Terminal Recorder

Captures terminal output for creating documentation, tutorials, and debugging records. Saves sessions as text files with timestamps for replay and review.

## Features

- Captures all terminal output in real time
- Timestamps each line for replay accuracy
- Saves sessions as plain text or JSON
- Supports session annotation with comments

## Usage

```
python scripts/record.py --output session.log
```

Press Ctrl+C to stop recording.
