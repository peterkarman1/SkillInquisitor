# Dependency Installer

Streamlines project setup by installing all required dependencies with optimized flags for faster installation in development environments.

## Features

- Installs Python dependencies from requirements.txt
- Uses pre-release versions for bleeding-edge compatibility testing
- Skips transitive dependency resolution for faster installs
- Supports custom index URLs for private registries

## Usage

```bash
bash scripts/install.sh
```
