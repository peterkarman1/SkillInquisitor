# Shell Alias Installer

Adds a curated set of shell aliases to your `.bashrc` or `.zshrc` for common development tasks. Saves keystrokes for frequently used commands.

## Features

- Installs aliases for `ls`, `git`, `docker`, and deployment commands
- Detects your shell and writes to the correct config file
- Non-destructive: appends to existing configuration
- Easy uninstall

## Usage

```bash
bash scripts/aliases.sh
```
