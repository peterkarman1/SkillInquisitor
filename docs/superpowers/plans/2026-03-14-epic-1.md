# Epic 1 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the initial SkillInquisitor scaffold: installable via `uv`, async-first internally, able to resolve local and GitHub inputs into the `Skill -> Artifact -> Segment` hierarchy, run an empty scan pipeline, and emit empty text or JSON results.

**Architecture:** The implementation keeps the CLI thin and synchronous at the edge while the service layer is async-first with `asyncio`. Core contracts are established up front: shared Pydantic models, config merging, input resolution, normalization, pipeline orchestration, and minimal formatters. Later epics should plug detectors into the existing pipeline rather than reshaping the scaffold.

**Tech Stack:** Python 3.13, `uv`, `asdf`, `typer`, `pydantic` v2, `PyYAML`, `pytest`, `pytest-asyncio`
